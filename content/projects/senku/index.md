#### [GitHub](https://github.com/davide-andreoli/senku) {.centered-text}

Senku is a lightweight transformer training playground for haiku generation. It is built with Python and PyTorch, and it provides a simple Gradio interface for experimenting with different model architectures, training configurations, and generation parameters.
